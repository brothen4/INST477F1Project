# INST477F1Project
For INST 477 F1 Project linked with API 

Usage
Selecting Season, Race, and Session: 

1. Start by selecting season year in drop down menu 
2. Select Grand Prix from drop down menu 
3. Select Session from drop down menu
4. Click "Load Session" 


Usage
Compare Telementry:

1. Select driver A and B that you are wanting to compare from drop menu
2. Select exact laps from list
3. Click "Compare Telementry"
